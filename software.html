My Software

HMM-FRAME
HMM-FRAME is designed to accurately locate and correct frameshift errors in next-generation sequencing (NGS) data using an augmented Viterbi algorithm on profile hidden Markov models (profile HMMs). By correcting the frameshift errors, it can classify more NGS sequences into their native protein domain families. Tool (Paper) (BibTex)

SAT-Assembler
SAT-Assembler is a scalable and accurate gene assembly tool for large-scale RNA_seq and metagenomic data. It recovers genes from gene families of particular interest to biologists with high coverage, low chimera rate, and extremely low memory usage compared with exiting gene aasembly tools. Moreover, it is naturally compitable with parallel computing platforms.

SALT
SALT is designed to sensitively and accurately classify short sequences into their native protein domain families with extremely low memory usage.

MetaDomain
MetaDomain is a protein domain classification tool designed for very short next-generation sequencing reads. It achieves better sensitivity and low false positive rate compared with the state-of-the-art profile hidden Markov model (profile HMM) alignment tool in identifying encoded domains from short sequences.
