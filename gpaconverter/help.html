GPA Converter 1.0

1. How to use this software
Enter the course name, credit and score for each course into the left spreadsheet. Then press the push button Calculate, your GPAs will be shown in the right table. The default number of rows for the left spreadsheet is 10. Use the button after the last row to add one row at a time if necessary. If you input invalid credits or scores a warning will be prompted.

2. How to input your score information
You can either input your score information into the left spreadsheet. Course name is optional.

3. How the GPAs are calculated
GPA is calculated using the following equation no matter what scale is used:

GPA = (score * credit) / (total credit)
