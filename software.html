My Software

HMM-FRAME
HMM-FRAME is designed to accurately locate and correct frameshift errors in next-generation sequencing (NGS) sequences using an augmented Viterbi algorithm on profile hidden Markov models. By correcting the frameshift errors, it can classify more NGS sequences into their native protein domain families
